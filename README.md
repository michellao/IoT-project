# IoT Project

## Installation

Brancher le **ventilateur** sur le pin **A4**

Brancher le **BME680** sur le pin **I2C_1**

## Capteurs

- Température
- Humidité

## Appareils contrôler

- Ventilateur

## Fonctions :
 - Controle manuel ou atomatique du ventilateur
 - Seuil de la température d'activation du ventilateur configurable
